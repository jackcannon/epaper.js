## Colors

| Color          | HEX Code  |
| -------------- | --------- |
| Gray 0 (White) | `#FFFFFF` |
| Gray 1         | `#C0C0C0` |
| Gray 2         | `#808080` |
| Gray 3 (Black) | `#000000` |

## Setup

1. `npm install`
2. `npm run start`
